Overview

Purpose

The purpose of this site is to provide the user with a clean and well organize overview of the company The Right Touch Painting. This site will also allow the user to contact and review the company.

Audience

This site is for those individuals who need someone to paint, power wash, drywall or frame for them.

Dynamic elements

The site will contain an image viewer for the different images of the work done previously and it will be able to send requests for free estinmates done for the company.
